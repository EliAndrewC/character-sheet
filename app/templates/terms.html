Terms of Service

Last updated: April 5, 2026

1. What This Is

L7R Character Builder is a small, invitation-only web application for managing character sheets in a homebrew Legend of the Five Rings tabletop RPG campaign. Access is limited to a whitelisted set of players personally invited by the game master (GM).

2. Accounts

You sign in using your Discord account. Only whitelisted Discord accounts may log in. We request only the minimal "identify" scope from Discord — we learn your Discord user ID and display name, nothing else. We do not access your Discord messages, friend list, or server memberships.

You may set a different display name within this application if you prefer not to use your Discord name.

3. Character Data

Character sheets you create are stored on our server. You retain ownership of your characters and may delete them at any time. The GM has administrative access to all character sheets for campaign management purposes.

Character sheets are publicly viewable. Anyone with the URL can view any character sheet without logging in. Only you, users you explicitly grant access to, and the GM can edit your character sheets.

4. Acceptable Use

This is a collaborative tool for our RPG group. Use it for its intended purpose. Don't attempt to access or modify other players' characters without permission.

5. No Warranty

This application is provided as-is, as a hobby project. We make no guarantees about uptime or data durability, though we do our best. Back up anything important to you.

6. Changes

These terms may be updated occasionally. Since we all know each other, we'll communicate any meaningful changes directly.
