# Combat vs non-combat XP - where the party spends its points

A question we've had for a while but never calculated: of the XP a character has
spent, how much went toward being good at *fighting* versus everything else -
courtly maneuvering, investigation, trade, lore, social standing? This is a
snapshot of where our ten player characters actually sit, taken from the live
character sheets, with some characters from past campaigns thrown in.

## What counts as "combat"

There is no perfectly clean line - almost any attribute helps *somewhere* - so
we made deliberate, consistent calls and lean toward counting things as combat
when they plausibly help in a fight. The rules:

- **Rings: all combat.** This is a simplification. Air on a character who never
  parries is mostly a social/initiative stat, and Void, Earth, etc. all have
  out-of-combat uses. But rings are too entangled with fighting to split
  cleanly, and every ring point also feeds Void points, which are combat fuel.
  So all ring XP is counted as combat.
- **School knacks: all combat.** Obvious for the pure-combat bushi schools.
  For the rest, every school's knacks and techniques still carry combat weight
  (for instance, all of the 5th Dan techniques function in a fight), so we
  treat *all* school expenditure - including knacks bought from another school -
  as combat rather than trying to prorate it.
- **Attack and Parry: always combat.**
- **General skills (the social and knowledge skills): non-combat by default.**
  The one exception is the skill that powers a school's own combat machinery.
  Most schools' 3rd Dan technique reads "...where X is your *attack* skill," so
  for them no general skill counts as combat. But some key off a general skill -
  Precepts (Brotherhood, Isawa Ishi, Kitsune Warden), Bragging (Ikoma Bard),
  Sincerity (Shosuro Actor) - and the Shosuro Actor's Special Ability keys off
  Acting. For those characters, that specific skill *is* a combat skill and its
  XP is counted as combat. (This is read straight out of the rules text, so it
  stays correct as schools change.)
- **Advantages: combat only if they have a combat application.** Lucky (reroll
  any roll), Strength of the Earth (a free raise on wound checks), and any
  Specialization in a combat skill all count as combat. Everything purely
  social, economic, or informational - Charming, Worldly, Tactician, Discerning,
  the Minor Clan allies, wealth, reputation - is non-combat.
- **Honor, Rank, and Recognition: non-combat.** These are pure status.

**A note on Quick Healer.** We count it as combat, but it is the most debatable
inclusion. You never use it *during* a fight - it only matters because it heals
the serious wounds combat inflicts, letting you back into the next fight sooner.
We treat that recovery as a combat investment (it has no use outside of being
hurt in a fight), the same way we count Strength of the Earth. If you would
rather call wound-recovery non-combat, it is worth 3 XP to the few characters
who took it and would nudge their shares down slightly.

**Disadvantages** are not placed in either bucket. The XP they hand back simply
lets a character buy more, so it already shows up inside the combat and
non-combat totals. The denominator for the percentages is total positive XP
spent (combat + non-combat), which equals base XP plus the XP gained from
disadvantages - so a character who took heavy disadvantages and spent it all has
a total above 200.

## Reading the results

The typical character spends about three-quarters of their XP on combat: the
median is 74.6% and the mean 73.2%. What is striking is how **tight the middle
of the party is** - six of the ten sit inside a 74.2%-to-79.6% band, barely five
points wide. Despite a Wasp bounty-hunter party drawn from wildly different
schools (a duelist, a shugenja, a scout, two actors, a courtier-bard...), most
of them have converged on roughly the same 75/25 combat-to-everything-else split.

The 37.8-point overall spread is driven almost entirely by three people at the
edges:

- **Tsuruchi Jimen (91%)** tops the list, but partly by construction: for a
  Shosuro Actor, Acting and Sincerity *are* combat skills (they fuel the school's
  attack/parry/wound-check dice and 3rd Dan), so his heavy investment in them
  lands in the combat column. His sheet is unusually pure - almost everything he
  bought feeds the fight.
- **Tsuruchi Sadakichi (53%)** and **Tsuruchi Shizukanaken (60%)** anchor the
  bottom. Both poured XP into social and knowledge skills that their schools do
  not turn into combat power - Sadakichi, a Mantis bushi, has the most
  non-combat XP of anyone (103). These are the party's "faces": still competent
  fighters, but deliberately built to operate out of combat as much as in it.

So the picture is a party with a strong, shared combat baseline (nobody is below
half, and the bulk cluster near 75%) and a small number of characters who have
consciously traded a slice of fighting power for breadth.

## Looking back: nine characters across past campaigns

The table above is a single moment in this party's life. A natural follow-up is
whether the combat share *moves* as a campaign runs - do players spend their
hard-won later XP the same way they spent their starting points, or does the
balance drift? We do not yet have a second dated snapshot of the live party, but
we do have something almost as good: nine PCs from *earlier* campaigns, each
recorded at both the **beginning** (their creation build) and the **end** of
their run. They come from two archives in `old-character-sheets/`:

- **Two full character sheets** (`*.odt`) - Kitsuki Tetsu and Wakuu. Stats were
  read by hand and re-costed through the same engine the live analysis uses (cost
  tables, 4th-Dan school-ring discount, auto-derived combat skill). See
  `combat_xp_past_campaigns.py`.
- **Seven XP ledgers** (`*Points.xls`) from a second player - Isamu, Naritsugu,
  Junichiro, Takumi, Masumune, Shori, Tozasu. These record one row per purchase
  with its *actual* XP cost, so nothing is re-costed; we read each number and
  bucket it. That neatly handles the fact that these campaigns used different
  conventions (rings maxing at 6, different free-ring placement, a peasant whose
  rings start at rank 1, a separate creation "character points" pool). See
  `combat_xp_past_ledgers.py`.

Every character is bucketed by the *same* combat-vs-non-combat rules as the live
party, including the auto-derived per-school combat skill (Investigation for
Kuni/Kitsuki, Sincerity for Merchant, Tact for Courtier, Precepts for the
Brotherhood; the pure bushi and the schoolless peasant have none).

| Character | School | Begin combat % | End combat % | Change |
|---|---|--:|--:|--:|
| Naritsugu | Kakita Duelist | 87.1% | 81.9% | -5.2 |
| Wakuu | Brotherhood of Shinsei Monk | 83.4% | 82.4% | -1.0 |
| Shori | Kitsuki Magistrate | 75.6% | 74.1% | -1.5 |
| Masumune | Mirumoto Bushi | 69.9% | (died at creation) | - |
| Isamu | Kuni Witch Hunter | 67.4% | 67.0% | -0.4 |
| Kitsuki Tetsu | Kitsuki Magistrate | 66.7% | 71.5% | +4.8 |
| Takumi | Courtier | 60.4% | 67.0% | +6.6 |
| Tozasu | none (peasant) | 59.5% | 65.4% | +5.9 |
| Junichiro | Merchant | 52.6% | 58.7% | +6.1 |

**The balance not only holds - it converges.** Two things happen at once across
these eight trajectories (Masumune died at creation, so he is a single point):

- The center barely moves. Mean combat share goes 69.1% -> 71.0%, median
  67.1% -> 69.3% - a campaign of growth shifts the typical character ~2 points.
- The *spread tightens*. At creation the eight range from 52.6% to 87.1% (a
  34.5-point gap); by the end they range from 58.7% to 82.4% (23.7 points). Every
  character who started above ~75% drifted **down** a little (Naritsugu -5.2,
  Shori -1.5, Wakuu -1.0), every character below ~67% climbed **up** 5-7 points
  (Junichiro +6.1, Takumi +6.6, Tozasu +5.9, Tetsu +4.8), and the ~67% middle
  (Isamu) stayed put.

In other words, players spend later XP a bit more like the party average than
their own starting build: the pure bushi eventually buys *some* breadth, and the
face/courtier eventually shores up their fighting. The combat/non-combat ratio is
mostly set on day one, but what drift there is pulls toward a shared ~70% center -
the same band the live Wasp party clusters in (mean 73.2%, median 74.6%). Across
both eras and both player groups, "about 70% combat" is the gravitational center.

### Per-source measurement notes and judgment calls

- **The .odt sheets spend zero XP on Honor / Rank / Recognition; the .xls ledgers
  spend it normally.** In Tetsu's and Wakuu's campaigns those advanced through
  play, not by XP - which is precisely what makes their hand-tracked totals
  reconcile (Wakuu to the exact XP at both ends; Tetsu within ~2%). The ledger
  characters bought HRR with XP, so it lands in their non-combat totals (as it
  does for the live party). Each is measured by what was actually spent.
- **Naritsugu's creation boundary.** His ledger front-loads 150 of creation
  "Earned", but per the player he was built on 100; we use 100, which moves the
  ~50 XP of purchases his budget could not yet afford (from Fire to 6 onward) into
  the campaign phase. That is why his *beginning* reads as an unusually pure 87% -
  he bought almost nothing but combat at creation and added the breadth later.
  Every other ledger's creation-earned is taken from the sheet, where it
  reconciles to the XP.
- **Shori replaced Masumune.** Masumune died at creation (his ledger has no
  post-creation "Earned" rows), so he is a single 69.9% data point. His cousin
  Shori was rebuilt on the XP Masumune had accumulated - and indeed Shori's
  creation total (324) lands right on Masumune's final total (319), an
  independent check on both parses. Worth noting: Masumune was a **Mirumoto
  Bushi**, a Void-ring dueling line, but invested heavily in **Fire** - an
  off-school ring - to be a better duelist, which is why his ledger pumps Fire
  rather than his school ring. It does not change his split (every ring is combat
  either way), but it is a good reminder that ring investment does not always
  follow the school ring.
- **Tozasu was a peasant with no school**, so he has no school knacks and no
  general combat skill; only rings, attack/parry, and combat advantages count as
  combat. "Deployment" on his sheet is an old name for Strategy, so it is
  non-combat like Strategy.
- **Borderline advantages** follow the live rules and the calls made for the
  .odt characters: Quick Healer, Strength of the Earth, Great Destiny, Lucky,
  Famous Sword, Reroll Wound Check, Avoid Death, and combat Specializations
  (Arrows, Ogres) count as combat; Discerning, Fierce, Tactician, Virtue,
  Imperial Favor, allies, reputations and the like are non-combat. Disadvantages,
  vows, and "Possession Points" are budget grants, not bucketed - same as the
  live analysis.

This is still a backward glance, not a forecast, but it is a stronger one than
two characters: across nine PCs from two separate campaigns it says the live
party's shares are unlikely to swing much, and that where a character sits is
mostly where they chose to sit on day one - with whatever drift there is bending
gently toward the party's ~70% center.

## Reproducing and updating this table

This reads the live campaign database, so it works in two steps: a **capture**
that freezes a dated snapshot (a committed JSON file in `combat_xp_snapshots/`),
and a **report** that builds the table from a snapshot. The report needs only
the committed snapshot, so anyone with a checkout can regenerate the table; the
script never holds any credential.

To take a fresh reading later in the campaign (e.g. after the party banks
another 100 XP), pull a local copy of the live DB with your own Fly login, then:

```sh
# capture a new dated snapshot from the local DB copy
DATABASE_URL=/tmp/l7r_live.db PYTHONPATH=. python3 \
    analysis/combat_vs_noncombat_xp.py capture

# regenerate this table + stats (paste stdout into the Results section above)
PYTHONPATH=. python3 analysis/combat_vs_noncombat_xp.py report
```

The full command for pulling the live DB, and every categorization rule, live in
the script's docstring. Each capture is kept, so over time we can watch the
balance move - `... report --date YYYY-MM-DD` re-runs any past snapshot, and
`... trend` prints one summary row per snapshot:

| Snapshot | PCs | Mean combat % | Median combat % | Range |
|---|--:|--:|--:|---|
| 2026-06-13 | 10 | 73.2% | 74.6% | 53.2% - 91.0% |

The "Looking back" section comes from two separate scripts, because those
characters predate the app and have no database row. Both reuse the same
categorization machinery, print their tables plus per-character change, and need
no database (the `.xls` reader needs `xlrd`: `pip install --break-system-packages
xlrd`):

```sh
# Kitsuki Tetsu + Wakuu, re-costed from hand-encoded .odt stats
PYTHONPATH=. python3 analysis/combat_xp_past_campaigns.py

# the seven *Points.xls XP ledgers (Isamu, Naritsugu, Junichiro, Takumi,
# Masumune, Shori, Tozasu) - costs read straight from each ledger row
PYTHONPATH=. python3 analysis/combat_xp_past_ledgers.py
```

## Caveats

- The biggest approximations are the two "all combat" rules (rings, and all
  school knacks). Both inflate the combat share somewhat for characters whose
  rings or school knacks have real out-of-combat use - but applying them
  uniformly keeps every character measured the same way.
- "Combat" here means *personal* combat effectiveness. A skill like Strategy
  (which Tactician boosts) matters in mass battle but does nothing for your
  attack roll, so it is counted as non-combat.
- A snapshot reflects each sheet's current built state at capture time, which
  would include any unpublished draft edits in progress. For the 2026-06-13
  snapshot every character was fully published with no pending edits, so the
  figures all reflect published builds.
